|---|---|---|---|

| Account not found | No account with that username | We couldn't find an account with that username. | ما لقينا حساب بهذا الاسم. |
| Wrong password | Wrong password | That password doesn't match. Please try again. | كلمة المرور غير صحيحة. حاول مرة ثانية. |
| Username taken | That username is taken — try another | This username is already in use. Try a different one. | اسم المستخدم هذا مستخدم بالفعل. جرّب اسم ثاني. |
| Invalid pickup code | Invalid or already used code | This code isn't valid anymore. Ask the donor to confirm the latest PIN/QR. | هذا الرمز غير صالح حالياً. اطلب من المتبرع تأكيد آخر PIN أو QR. |
| Different business | This claim is for a different business | This pickup code belongs to another donor location. | رمز الاستلام هذا يخص جهة تبرع أخرى. |

| Create step 3 | Pickup window is too short | Please choose a longer pickup window so someone can reach you. | من فضلك اختر وقت استلام أطول حتى يتمكن أحد من الوصول إليك. |
